# CT100 — VSCode Server
Entorno de desarrollo remoto accesible desde el navegador.

## Recursos asignados
- OS: Ubuntu 24.04
- CPUs: 2 cores
- RAM: 4 GB
- Disco: 100 GB (local-lvm)
- Red: vmbr0, IP estática en red local

## Descripción
VSCode Server permite acceder a un entorno de desarrollo completo
desde cualquier navegador, sin instalar nada localmente.
Autenticación mediante token seguro.
